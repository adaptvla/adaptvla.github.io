AdaptVLA: Training-Free VLA Test-Time Adaptation via Language Correction

Owen Kwon1
1Carnegie Mellon University

Abstract

Vision-Language-Action (VLA) models show strong real-world performance but suffer from action miscalibration when deployed in new environments. We propose AdaptVLA, a training-free wrapper that corrects VLA actions at deployment time using minimal language feedback.

Our method enables immediate policy steering without retraining by applying additive bias corrections based on human language instructions. We demonstrate 100% success rate on all failure cases across LIBERO benchmarks with just 1-2 corrections per task, achieving 30-40% task time reduction with sub-millisecond overhead.

The Problem

While VLAs handle diverse manipulation challenges, they exhibit zero-shot performance limitations when deployed in new settings due to distribution mismatch between training and deployment environments.

Traditional adaptation methods require:

  • Resource-intensive data collection
  • Complex fine-tuning processes
  • High-quality teleoperated demonstrations

Question: Can we immediately steer VLA policy using minimum resources at deployment?

Our Approach

Key Insights

  1. Additive Correction Model: Robot actions can be calibrated by applying additive bias corrections
  2. Self-Correction Propagation: Once initial actions are calibrated, the VLA automatically generates correct subsequent actions without further intervention
  3. Language as Interface: Human corrections via natural language feedback mapped to action space parameters

Method Pipeline

  1. Robot Execution: Deploy VLA policy in new environment
  2. Human Review: Observe trajectory and provide language correction (e.g., "move left more in 1-2s")
  3. Language-Action Mapping: LLM converts feedback to structured corrections (axis, magnitude, time window)
  4. Action Calibration: Apply time-varying corrections with confidence weighting

Key Findings

Self-Correction Propagation

A few initial corrections automatically steer the entire trajectory. After calibrating just a few timesteps, the VLA generates significantly different subsequent actions, enabling task success without continuous human intervention.

Minimal Human Burden

  • Only 1-2 language corrections needed per task
  • No teleoperation required
  • No model retraining
  • Real-time deployment (<1ms overhead)
